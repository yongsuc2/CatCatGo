## 사용법

1. [공통_스타일_가이드.md](공통_스타일_가이드.md)에서 프롬프트 조합법 확인
2. 해당 캐릭터 문서에서 캐릭터 프롬프트 확인
3. `generate_sprite.py` 스크립트로 자동 생성, 또는 ComfyUI에 수동 입력
4. 캐릭터당 1장 생성 (`sprite.png`, 512x512 PNG, 투명 배경)
5. 생성된 파일을 `Assets/_Project/Resources/Chars/{id}/sprite.png`에 저장

## 저장 규칙

```
Chars/{id}/
├── sprite.png    (단일 스프라이트, 512x512 PNG, RGBA)
└── ref.*         (선택, IP-Adapter 참조 이미지)
```
