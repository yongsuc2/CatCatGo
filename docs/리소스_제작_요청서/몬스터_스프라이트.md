# 몬스터 스프라이트 제작 요청서

레퍼런스: 플레이어 캐릭터(주황 고양이)와 동일한 아트 스타일.
스타일 상세: [공통_스타일_가이드.md](공통_스타일_가이드.md) 참조.

## 생성 스크립트

| 용도 | 스크립트 |
|------|---------|
| 정적 스프라이트 (ComfyUI) | `C:\Cat\ComfyUI\scripts\generate_sprite.py` |
| 프레임 애니메이션 (Grok SDK) | `scripts\generate_animation.py` |

## 출력 포맷

### 정적 스프라이트

- 캐릭터당 1장: `Chars/{id}/sprite.png` (512x512 PNG, RGBA, 투명 배경)
- 포즈: idle (정지, 편안한 자세)

### 프레임 애니메이션

```
Chars/{id}/
├── side.jpg                 (사이드뷰 참조 이미지)
├── idle.mp4 / walk.mp4 / attack.mp4  (동영상 소스)
├── idle/frame_0001.png ~    (투명 프레임, 12장)
├── walk/frame_0001.png ~    (투명 프레임, 12장)
└── attack/frame_0001.png ~  (투명 프레임, 16장)
```

- 사이드뷰: 3/4 앵글, 적은 왼쪽 향함, 초록 배경(#00FF00)
- 동영상 → 프레임 추출 → 배경 투명화 (extract_frames.ps1)
- 상세: animation-pipeline skill 참조

## 참고

- 정확한 프롬프트는 `generate_sprite.py`의 `MONSTERS` dict에 정의
- 각 캐릭터의 seed 값도 스크립트에서 관리
- 재생성 시 seed 변경으로 다른 결과 유도 가능
