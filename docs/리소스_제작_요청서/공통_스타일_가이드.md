# 공통 스타일 가이드

모든 캐릭터/몬스터 리소스에 적용되는 아트 스타일 규칙.

## 핵심 키워드

**덩어리감 / 매트 캐주얼 토이 / 평면 색면 / 굵은 외곽선 / 디테일 최소화**

## 조형 원칙

### 실루엣

- 몸통은 콩/감자 형태의 큰 덩어리 1~2개로 구성
- 실루엣만으로 캐릭터가 읽혀야 함

### 비율 (2등신 SD)

| 부위 | 비율 |
|------|------|
| 머리 | 55~75% (전체 인상 지배) |
| 몸 | 25~45% |
| 목 | 거의 없음 (머리 → 바로 몸) |

### 팔/다리

- 팔: 소시지(원통) 1개 + 손(둥근 물방울) 1개, 짧고 통통
- 다리: 발바닥 포함 두꺼운 짧은 기둥 2개
- 손: 장갑 낀 앞발(미튼), 손가락 표현 없음

## 쉐이딩 — 2톤 룰

| 단계 | 설명 |
|------|------|
| 기본색 | 밝은 면 |
| 그림자 | 한 단계 어두운 면, 큰 덩어리로 툭 얹기 |
| 하이라이트 | (선택) 점/작은 면 1개 |

- 그라데이션 거의 사용하지 않음
- 재질감: 매트(무광) + 말랑한 듯 단단한 캐주얼 토이

## 디테일 최소화

- 줄무늬/털 무늬: 얼굴 1포인트 + 몸 0~1포인트만
- 하이라이트: 점/작은 면 1개만 (플라스틱 스펙큘러 금지)
- 털 결/재질 텍스처 거의 없음
- 얼굴: 눈/코/입이 작고 단순 (점, 짧은 선, 작은 U자)

## 외곽선

- 굵고 균일한 외곽선 → "장난감 같은 덩어리" 인상
- 각진 하이라이트/강한 광택 배제
- 굵은 외곽선 + 평면 색면 중심

## ComfyUI 스프라이트 규격

| 항목 | 값 |
|------|-----|
| 해상도 | 512 x 512 px |
| 배경 | 투명 (PNG, RGBA) |
| 포즈 | idle (정지, 편안한 자세) |
| 출력 | 캐릭터당 1장 (`sprite.png`) |

## 프롬프트 조합법

ComfyUI positive prompt = `{STYLE_PREFIX}{캐릭터 설명}, {STYLE_SUFFIX}no text, no labels, masterpiece, best quality`

### STYLE_PREFIX (모든 프롬프트 앞에)

```
chibi, cel shaded, thick black outlines, flat cel-shading, solid pure white background, centered composition, solo,
```

### STYLE_SUFFIX (모든 프롬프트 뒤에)

```
2d illustration, simple design, one character only, full body, standing still, relaxed idle pose,
```

### 네거티브 프롬프트 (공통)

```
realistic, photograph, 3d render, heavy gradient shading, strong specular highlights, complex details, thin outlines, realistic proportions, text, labels, numbers, grid lines, background scenery, multiple characters, two characters, group, crowd, duo, pair, many, several, off-center, cropped, character sheet, multiple views, turnaround, sprite sheet, collage, grid, reference sheet, tiled, repeated, pattern, circle background, round frame, vignette
```

## 조합 예시

아메바(amoeba) 스프라이트를 생성하는 경우:

- **Positive**: `chibi, cel shaded, thick black outlines, flat cel-shading, solid pure white background, centered composition, solo, a single cute chibi amoeba blob creature, slime body, one big eye, purple-pink color, 2d illustration, simple design, one character only, full body, standing still, relaxed idle pose, no text, no labels, masterpiece, best quality`
- **Negative**: (공통 네거티브)
- **Resolution**: 512 x 512

## 모션

### Transform 애니메이션 (정적 스프라이트 기반)

스프라이트 1장에 Unity Transform 애니메이션 적용:
- **idle**: bob (상하 4px, 2.5Hz) + breathe (scale 97~103%, 2Hz)
- **attack**: 접근 → squash-stretch 히트 임팩트 → 후퇴

### 프레임 애니메이션 (동영상 기반)

Grok SDK로 동영상 생성 → 프레임 추출 → 투명 PNG 시퀀스:
- **idle**: 4프레임 (2초, 2fps), 호흡/상하 움직임
- **walk**: 6프레임 (3초, 2fps), 걷기 사이클
- **attack**: 6프레임 (3초, 2fps), 공격 동작
- 사이드뷰 이미지: 896x1280px (aspect_ratio 3:4, resolution 1k)
- 동영상: 480p, 9:16
- 사이드뷰 3/4 앵글, 초록 배경(#00FF00) → floodfill 배경 제거
- 생성 스크립트: `scripts/generate_animation.py`
- 상세: animation-pipeline skill 참조
