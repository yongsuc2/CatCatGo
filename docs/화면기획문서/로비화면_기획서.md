# 로비 화면 기획서

---

## 1. 화면 구성 (위 -> 아래)

### 1-1. 화면 헤더
```
[☰]  카피바라 고!
```
- 좌측 상단 `[☰]` 메뉴 버튼 클릭 시 **메뉴 목록 팝업** 오픈
- 타이틀 중앙 정렬 유지 (우측에 동일 크기 spacer)

### 1-2. 플레이어 스탯 카드
```
┌────────────────────────────────┐
│ 플레이어 스탯          [상세] │
│ 체력      {hp} / {maxHp}      │
│ 공격력    {atk}               │
│ 방어력    {def}               │
│ 치명타    {crit}%             │
└────────────────────────────────┘
```
- `[상세]` 버튼 클릭 시 **상세 스탯 팝업** 오픈
- 스탯은 `Player.computeStats()` 최종 합산값

### 1-3. 상세 스탯 팝업 (modal-overlay)
```
┌────────────────────────────────┐
│          상세 스탯              │
│                                │
│ ── 기본 스탯 ──               │
│ 최대 체력          1,250       │
│   기본               100       │
│   재능               450       │
│   장비               500       │
│   유산                50       │
│   펫                150       │
│                                │
│ 공격력               320       │
│   기본                10       │
│   재능               120       │
│   장비               140       │
│   펫                 50       │
│                                │
│ 방어력                85       │
│   기본                 5       │
│   재능                40       │
│   장비                40       │
│                                │
│ 치명타 확률          0.0%      │
│                                │
│ ── 전투 스탯 ──               │
│ 치명타 데미지        150%      │
│ 흡혈률               0.0%      │
│ 회피율               0.0%      │
│ 반격 확률            0.0%      │
│                                │
│ [          닫기          ]     │
└────────────────────────────────┘
```
- 팝업 외부(overlay) 클릭 시 닫힘
- 기본 스탯: 출처별 분해 표시 (기본/재능/장비/유산/펫)
- 0인 출처는 생략
- 전투 스탯: 항상 표시 (0이어도 향후 확장 대비)

### 1-4. 정보 카드
```
┌────────────────────────────────┐
│ 재능 등급     {서브등급 라벨}  │
│ 클리어 챕터     {chapterMax}   │
│ 탑 층수         {floor}-{stg} │
│ 아레나 티어     {arenaTier}    │
└────────────────────────────────┘
```

### 1-5. 메뉴 그리드 (2x2)
```
┌──────────┐ ┌──────────┐
│  ⚔️ 모험  │ │  🏰 콘텐츠│
│ 스태미나:N │ │ 탑/던전   │
└──────────┘ └──────────┘
┌──────────┐ ┌──────────┐
│  📈 성장  │ │  🎁 뽑기  │
│ 재능/유산 │ │ 보석: N   │
└──────────┘ └──────────┘
```

### 1-6. 메뉴 목록 팝업 (MenuListPopup)
```
┌────────────────────────────────┐
│ 메뉴                      [X] │
│                                │
│ ┌────────────────────────────┐ │
│ │          가방              │ │
│ └────────────────────────────┘ │
│ ┌────────────────────────────┐ │
│ │          우편              │ │
│ └────────────────────────────┘ │
│ ┌────────────────────────────┐ │
│ │         퀘스트             │ │
│ └────────────────────────────┘ │
│ ┌────────────────────────────┐ │
│ │        출석체크            │ │
│ └────────────────────────────┘ │
│ ┌────────────────────────────┐ │
│ │          설정              │ │
│ └────────────────────────────┘ │
└────────────────────────────────┘
```
- 각 항목 클릭 시 해당 기능으로 이동
- 가방 → InventoryPopup (소모성 아이템 수량 표시)
- 우편 → MailPopup (우편함)
- 퀘스트 → QuestScreen 화면 이동
- 출석체크 → EventScreen 화면 이동
- 설정 → SettingsPopup (배경음/효과음 토글)

### 1-7. 가방 팝업 (InventoryPopup)
```
┌────────────────────────────────┐
│ 가방                      [X] │
│                                │
│ 골드              1,234        │
│ 보석                 50        │
│ 스태미나          80/100       │
│ 도전 토큰            3         │
│ 곡괭이               2         │
│ 장비석              15         │
│ 파워 스톤            5         │
│ 펫 알                2         │
│ 펫 사료              8         │
│ 스컬 서적            1         │
│ 나이트 서적          0         │
│ 레인저 서적          0         │
│ 고스트 서적          0         │
└────────────────────────────────┘
```
- Player.Resources에서 전 항목 수량 표시
- 스크롤 가능

### 1-8. 우편 팝업 (MailPopup)
```
┌────────────────────────────────┐
│ 우편                      [X] │
│                                │
│       우편함이 비어 있습니다.   │
│                                │
└────────────────────────────────┘
```

### 1-9. 설정 팝업 (SettingsPopup)
```
┌────────────────────────────────┐
│ 설정                      [X] │
│                                │
│ 배경음              [ON/OFF]   │
│ 효과음              [ON/OFF]   │
└────────────────────────────────┘
```
- 토글 상태 PlayerPrefs에 저장
- 기존 SettingsScreen(저장/불러오기/내보내기)과는 별도

## 2. 상호작용

| 요소 | 액션 | 결과 |
|------|------|------|
| [☰] 메뉴 버튼 | 클릭 | 메뉴 목록 팝업 오픈 |
| [상세] 버튼 | 클릭 | 상세 스탯 팝업 오픈 |
| 팝업 overlay | 클릭 | 팝업 닫힘 |
| [닫기] 버튼 | 클릭 | 팝업 닫힘 |
| 모험 카드 | 클릭 | ChapterScreen 이동 |
| 콘텐츠 카드 | 클릭 | ContentScreen 이동 |
| 성장 카드 | 클릭 | TalentScreen 이동 |
| 뽑기 카드 | 클릭 | GachaScreen 이동 |
| 메뉴 > 가방 | 클릭 | InventoryPopup 오픈 |
| 메뉴 > 우편 | 클릭 | MailPopup 오픈 |
| 메뉴 > 퀘스트 | 클릭 | QuestScreen 이동 |
| 메뉴 > 출석체크 | 클릭 | EventScreen 이동 |
| 메뉴 > 설정 | 클릭 | SettingsPopup 오픈 |

## 3. 데이터 소스

| UI 요소 | 데이터 소스 |
|---------|------------|
| 기본 스탯 분해 | `Player.getStatsBreakdown()` |
| 전투 스탯 | `Player.getCombatPassives()` |
| 치명타 데미지 | `BattleDataTable.damage.critMultiplier` |
