# 플레이어 캐릭터 애니메이션 스펙

## 개요

플레이어 캐릭터는 적 몬스터와 다른 고품질 애니메이션 스펙을 적용한다.
적은 4~6프레임이지만, 플레이어는 24프레임으로 부드러운 모션을 제공한다.

## 적 vs 플레이어 비교

| 항목 | 적 (몬스터) | 플레이어 |
|------|-----------|---------|
| 바라보는 방향 | 왼쪽 (facing left) | 오른쪽 (facing right) |
| 동영상 길이 | 2~3초 | 6초 |
| 추출 fps | 2 | 4 |
| idle 프레임 수 | 4 | 24 |
| walk 프레임 수 | 6 | 24 |
| attack 프레임 수 | 6 | 24 |
| 동영상 해상도 | 480p | 480p |
| 동영상 비율 | 9:16 | 9:16 |
| 프레임 크기 | 480x848 px | 480x848 px |
| 동영상당 비용 | $0.10~$0.15 | $0.30 |
| 총 비용 (동영상) | $0.40 | $0.90 |

## 캐릭터 설명

```
a cute chibi cat character, round chubby cream colored body, big round head,
small dot eyes, simple mouth, whiskers, two short stubby arms on sides,
short stubby legs, pink toe beans, brown hood with cat ears,
gold star brooch on chest, small tail
```

## 리소스 경로

```
Assets/_Project/Resources/Chars/player/
├── ref.jpg           (컨셉 참고 이미지)
├── side.jpg          (사이드뷰, 오른쪽 향함)
├── idle.mp4          (6초)
├── walk.mp4          (6초)
├── attack.mp4        (6초)
├── idle/
│   ├── frame_0001.png ~ frame_0024.png  (480x848, 투명 배경)
├── walk/
│   ├── frame_0001.png ~ frame_0024.png
└── attack/
    ├── frame_0001.png ~ frame_0024.png
```

## 생성 스크립트

```bash
python scripts/generate_player_animation.py                    # 전체 파이프라인
python scripts/generate_player_animation.py --step image       # 사이드뷰 이미지만
python scripts/generate_player_animation.py --step video       # 동영상만 (side.jpg 필요)
python scripts/generate_player_animation.py --step frames      # 프레임 추출만 (mp4 필요)
python scripts/generate_player_animation.py --step resize      # 프레임 480x848 리사이즈
```

## 프롬프트

### 사이드뷰 이미지

```
Side view of {캐릭터 설명}, facing right,
3/4 angle side view with features slightly visible,
chibi 2-head-tall SD proportions, cel-shaded, thick black outlines, flat solid colors,
toy-like matte appearance, minimal detail, full body, single character only,
on a solid bright green (#00FF00) background,
no text, no labels, clean illustration, masterpiece, best quality
```

### idle 동영상

```
Animate this exact character with a gentle idle breathing animation, facing right.
Subtle rhythmic breathing - body slightly expands and contracts, small gentle bobbing.
The first frame and last frame must look identical for a perfect seamless loop.
Keep the character exactly identical to the reference image in every detail.
Maintain the solid bright green background throughout.
Smooth looping animation. Side view maintained. No audio, no sound, no music.
```

### walk 동영상

```
Animate this exact character walking in place (treadmill style), facing right.
Arms and legs move in a natural walking rhythm.
Keep the character exactly identical to the reference image in every detail.
Maintain the solid bright green background throughout.
Smooth looping animation. Side view maintained. No audio, no sound, no music.
```

### attack 동영상

```
Animate this exact character performing a quick attack motion toward the right.
Wind up, then strike forward with force, then return to ready pose.
Keep the character exactly identical to the reference image in every detail.
Maintain the solid bright green background throughout.
Side view maintained. No audio, no sound, no music.
```

## 인게임 재생 스펙

| 항목 | 값 |
|------|-----|
| idle 재생 fps | 8 |
| attack 재생 fps | 12 |
| idle | 루프 재생 |
| attack | 1회 재생 후 idle 복귀 |
| 표시 크기 | 프레임 비율 기반 자동 계산 (너비 135px 기준) |

## 배경 제거

- 배경색: `#00FF00` (크로마키 그린)
- 도구: ImageMagick floodfill (4 corners)
- Fuzz: 25%
- 동영상 인코딩 후 색상 변화(예: `#46C45C`)는 fuzz로 처리
