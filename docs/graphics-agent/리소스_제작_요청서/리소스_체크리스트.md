### 리소스 포맷

| 항목 | 값 |
|------|-----|
| 파일 | `Chars/{id}/sprite.png` |
| 해상도 | 512x512 PNG, RGBA |
| 포즈 | idle (정지) |
| 배경 | 투명 |
