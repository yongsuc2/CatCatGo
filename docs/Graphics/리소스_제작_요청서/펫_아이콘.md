# 펫 아이콘 제작 요청서

## 개요

| 항목 | 값 |
|------|-----|
| 총 수량 | 18종 (S 9종, A 6종, B 3종) |
| 해상도 | 64x64 PNG, RGBA |
| 배경 | 투명 |
| 스타일 | [공통_스타일_가이드.md](공통_스타일_가이드.md) 참조 |
| 저장 경로 | `Assets/_Project/Resources/Icons/pet/` |
| 파일 명명 | `pet_{id}.png` (예: `pet_elsa.png`) |
| 로드 방식 | SpriteDatabase Inspector에서 할당, 또는 Resources 자동 로드 |

## 아이콘 특성

- 정면 또는 3/4 앵글의 얼굴/상반신 클로즈업
- 64x64 소형이므로 실루엣과 색상으로 즉시 구별 가능해야 함
- 공통 스타일: 치비, 두꺼운 외곽선, 평면 셀 셰이딩, 매트 토이 질감
- 몬스터 스프라이트와 동일한 아트 톤 유지

## 프롬프트 조합법

ComfyUI positive prompt = `{ICON_PREFIX}{캐릭터 설명}, {ICON_SUFFIX}`

### ICON_PREFIX (모든 프롬프트 앞에)

```
chibi, cel shaded, thick black outlines, flat cel-shading, solid pure white background, centered composition, solo, close-up portrait,
```

### ICON_SUFFIX (모든 프롬프트 뒤에)

```
2d illustration, simple design, one character only, head and upper body, facing viewer, cute expression, no text, no labels, masterpiece, best quality
```

### 네거티브 프롬프트 (공통)

```
realistic, photograph, 3d render, heavy gradient shading, strong specular highlights, complex details, thin outlines, realistic proportions, text, labels, numbers, grid lines, background scenery, multiple characters, full body, legs, feet, off-center, cropped, character sheet, multiple views
```

## 조합 예시

엘사(elsa) 아이콘을 생성하는 경우:

- **Positive**: `chibi, cel shaded, thick black outlines, flat cel-shading, solid pure white background, centered composition, solo, close-up portrait, a cute chibi ice fairy cat, ice blue and silver white fur, frosty sparkles, small ice crystal tiara on head, gentle blue eyes, chilly aura, 2d illustration, simple design, one character only, head and upper body, facing viewer, cute expression, no text, no labels, masterpiece, best quality`
- **Negative**: (공통 네거티브)
- **Resolution**: 512 x 512 (생성 후 64x64로 리사이즈)

## 생성 후 처리

1. ComfyUI로 512x512 생성
2. 배경 투명 처리 (배경이 흰색인 경우)
3. 64x64로 리사이즈 (Lanczos 리샘플링 권장)
4. `Assets/_Project/Resources/Icons/pet/pet_{id}.png`에 저장
5. SpriteDatabase Inspector에서 해당 펫에 아이콘 할당
