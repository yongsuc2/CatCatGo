# BUG-005: 일일 퀘스트 2개 누락 (아레나, 여행)

## 상태
Closed

## 심각도
Major

## 발견일
2026-03-07

## 발견 경위
consistency-report-v2 작성 중, 07_재화시스템.md의 일일 퀘스트 5개 명세와 quest.data.json의 3개 데이터를 비교하여 발견.

## 증상
기획서(07_재화시스템.md)에 일일 퀘스트 5개가 정의되어 있으나, quest.data.json에는 3개만 구현됨.

**기획서 명세 (5개):**
1. 챕터 3회 클리어 -> 보석 30 (구현됨)
2. 던전 3회 완료 -> 골드 450 (구현됨)
3. 탑 2회 도전 -> 장비석 3 (구현됨)
4. **아레나 1회 전투 -> 보석 20** (누락)
5. **여행 5회 -> 골드 350** (누락)

**quest.data.json (3개만 존재):**
- daily_chapter, daily_dungeon, daily_tower

## 원인
아레나(PvP)와 탐방(Travel) 기능이 미구현 상태이므로, 해당 기능에 연동된 퀘스트도 데이터에서 누락된 것으로 추정.

## 영향 범위
- 현재 게임 동작에는 직접적 영향 없음 (미구현 기능의 퀘스트)
- 기획서와 데이터 간 불일치 상태 유지
- 향후 아레나/탐방 구현 시 퀘스트 데이터 추가 필요
- 기획서에 "미구현" 표기가 없어 구현 상태를 오해할 수 있음
