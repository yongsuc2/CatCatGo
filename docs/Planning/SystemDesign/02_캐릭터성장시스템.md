# 카피바라고! 역기획서 - 캐릭터 성장 시스템

## 개요

카피바라고는 다수 영웅 수집이 아닌, **단일 카피바라 캐릭터의 영구 성장** 구조를 채택한다.
성장은 크게 **특성(Talent)** 과 **전승(Heritage)** 두 축으로 나뉜다.

## 특성(Talent) 시스템

### 강화 대상

| 스탯 | 강화 우선순위 | 이유 |
|------|-------------|------|
| ATK | 1순위 | 킬 속도 = 생존력 (빠른 처치가 피해 감소) |
| HP | 2순위 | 생존 턴 수 확보 |
| DEF | 3순위 | ATK/HP 대비 효율 낮음 |

### 레벨당 스탯 증가

수치: `talent.data.json`의 `statPerLevel` 참조

### 강화 비용

- 재화: 골드
- **서브 등급(단) 내에서는 비용 고정**, 다음 서브 등급으로 전환 시에만 비용 상승
- 3개 스탯 모두 동일 비용 (총 레벨 기반)
- 비용 공식: `floor(baseCost × costGrowth^(tier-1))`
- 구체적 수치는 `talent.data.json`의 `gradeConfig` 참조

### 등급 구조

6개 메인 등급, 각 등급을 서브 등급(단)으로 세분화:

```
수련생 → 모험가 → 정예 → 달인 → 전사 → 영웅
```

등급별 서브등급 수, 총 레벨, 레벨 범위는 `talent.data.json`의 `gradeConfig` 참조.

- 각 서브 등급 = 30레벨 (ATK 10 + HP 10 + DEF 10)
- UI 표시: "모험가 2단" 형식

### 서브 등급 진급

- 각 스탯은 서브 등급 내에서 최대 10레벨
- 3개 스탯 모두 10 도달 시 → 서브 등급 진급 + 3개 스탯 0으로 리셋
- 서브 등급 구조: `talent.data.json`의 `gradeConfig`에서 자동 생성

### 서브 등급 전환 보너스

모든 서브 등급 전환 시 ATK/DEF 보너스가 자동 적용된다 (첫 번째 서브 등급 제외).

**두 종류의 보너스**:

1. **메인 등급 전환 보너스** — 새 메인 등급 1단 진입 시 일회성 대량 보너스
   - 수치: `talent.data.json`의 `mainGradeBonuses` 참조

2. **서브 등급 전환 보너스** — 그 외 모든 서브 등급 전환 시 ATK/DEF 교대 적용
   - 보너스 공식: `tierBonusBase + subTierIdx × tierBonusGrowth`
   - 등급 내에서 점진적으로 증가
   - DEF/ATK가 번갈아가며 적용 (bonusIdx 기반, 짝수=DEF, 홀수=ATK)
   - 수치: `talent.data.json`의 `gradeConfig.tierBonusBase/tierBonusGrowth` 참조

### 마일스톤 보상

총 레벨 기준으로 일정 간격마다 보상이 배치된다.

- **간격**: 10레벨마다 (서브 등급당 마일스톤 2개: +10, +20 위치)
- **제외**: 서브 등급 경계(30의 배수)에는 마일스톤 미배치
- **보상 종류**: 골드 일시 지급과 골드 획득량 % 증가가 **교대**로 지급
  - 짝수 번째: 골드 = `해당 레벨 업그레이드 비용 × goldCostMultiplier`
  - 홀수 번째: 골드 획득량 +`goldBoostPercent`%
- **수령 방식**: 도달 시 재능 화면에서 수동 수령 (개별 클릭 또는 "모두 수령")
- **소급 수령**: 이미 지나간 마일스톤도 소급 수령 가능
- **골드 획득량 증가**: 챕터 내 전투/인카운터/클리어 골드에 적용
- 수치: `talent.data.json`의 `milestoneConfig` 참조

### 재능 화면 UI 구성

1. **등급 정보 카드**: 현재 서브 등급, 서브 등급 내 진행도, 총 레벨/다음 메인 등급 목표, 골드 획득량 보너스
2. **등급 보상 트랙**: 현재 레벨 기준 전후 15레벨 범위의 슬라이딩 윈도우로 마일스톤/전환 노드 표시
3. **스탯 강화 카드 3개**: ATK/HP/DEF 각각 서브 등급 내 레벨, 레벨당 증가량, 비용, 강화 버튼
4. **전승 카드**: 영웅 등급 도달 시에만 표시

## 전승(Heritage) 시스템

### 해금 조건
- 특성이 **"영웅" 등급**에 도달 시 해금

### 계승 루트 (5종)

| 계승 | 특화 빌드 | 핵심 전략 | 전용 재화 |
|------|-----------|-----------|-----------|
| 해골 | 연타/검기 | 다회 타격으로 높은 DPS | 뼈 책 |
| 기사 | 반격/광창 | 피격 시 반격으로 안정적 딜 | 기사 책 |
| 유협 | 치명타/수리검 | 크리티컬 기반 폭딜 | 레인저 책 |
| 유령 | 스킬/속성 | 속성 스킬 시너지 극대화 | 영혼 책 |
| 드래곤 | 미출시 | - | - |

### 전승 강화 구조

```
[전용 재화(책) 투입] → [전승 레벨 상승] → [빌드 특화 패시브 강화]
                                              ↓
                                    [해당 빌드 스킬 효과 증폭]
```

### 계승 루트별 시너지 스킬 매핑

#### 해골 계승
- 연타 마스터리 → 공격 시 확률로 추가 타격
- 검기 → 추가 범위 공격

#### 기사 계승
- 반격 → 피격 시 자동 공격
- 광창 → 관통 공격

#### 유협 계승
- 치명타 마스터리 → 크리티컬 확률/배율 증가
- 수리검 → 추가 투사체 공격

#### 유령 계승
- 속성 스킬 전반 → 속성 데미지 강화
- 스킬 강화 계열 → 스킬 발동 효과 증폭

## 성장 자원 흐름도

```
[골드]
  ├→ 특성 강화 (ATK/HP/DEF 레벨업)
  └→ 전승 강화 (계승 루트 레벨업)

[전용 책]
  └→ 전승 강화 (계승별 전용 재화)

[장비 강화석]
  └→ 장비 시스템 (별도 문서)

[펫 사료/알]
  └→ 펫 시스템 (별도 문서)
```
