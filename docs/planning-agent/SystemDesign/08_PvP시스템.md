# 카피바라고! 역기획서 - PvP 시스템 (경기장)

## 개요

| 항목 | 내용 |
|------|------|
| 명칭 | 경기장 (Arena) |
| 유형 | 비동기 PvP (실시간 대전 아님) |
| 매칭 | 같은 티어 유저 4명 매칭 |
| 진입 비용 | 입장권 |
| 추가 도전 | 보석으로 재도전 가능 |

## 티어 체계

```
브론즈 → 실버 → 골드 → 플래티넘 → 다이아 → 마스터
```

| 티어 | 승급 조건 | 보상 수준 |
|------|-----------|-----------|
| 브론즈 | 초기 배치 | 기본 |
| 실버 | 브론즈 상위 랭크 | 낮음 |
| 골드 | 실버 상위 랭크 | 중간 |
| 플래티넘 | 골드 상위 랭크 | 중상 |
| 다이아 | 플래티넘 상위 랭크 | 높음 |
| 마스터 | 다이아 상위 랭크 | 최상 |

## 전투 구조

```
[입장권 소모] → [같은 티어 유저 4명 매칭]
    ↓
[자동 전투] → 플레이어 빌드 기반 자동 교전
    ↓
[순위 결정] → 4명 중 순위 산정
    ↓
[보상 지급] → 순위에 따른 보상
    ↓
[티어 포인트 변동] → 승급/강등 반영
```

## 전투 메커니즘

| 항목 | 내용 |
|------|------|
| 전투 방식 | 턴제 자동 전투 (PvE와 동일) |
| 적용 스탯 | 특성 + 장비 + 전승 (영구 성장만 적용) |
| 스킬 | 챕터 스킬 미적용, 장비/전승 효과만 |
| 승리 기준 | 상대 HP 0 |
