# CatCatGo 밸런스 데이터 시트

> 모든 밸런스 수치는 데이터 테이블(`Assets/_Project/Data/Json/`)에서 관리한다.
> 이 문서는 각 시스템의 밸런스 데이터가 어디에 정의되어 있는지 안내하는 색인이다.
> **구체적 수치는 이 문서에 기재하지 않으며, 반드시 데이터 테이블 원본을 확인할 것.**

---

## 1. 전투 메카닉

> `battle.data.json`

### 데미지 공식

물리/마법/고정 데미지 공식의 모든 계수는 `battle.data.json`의 `damage` 섹션에서 관리한다.

| 항목 | 데이터 경로 | 설명 |
|------|-----------|------|
| 물리 방어 상수 (k) | `damage.defenseConstant` | `k/(k+DEF)` 공식 적용 |
| 마법 방어 상수 (k_m) | `damage.magicDefenseConstant` | `k_m/(k_m+DEF)` 공식 적용 |
| 기본 마법 계수 | `damage.baseMagicCoefficient` | magic_mastery 패시브로 가산 |
| 반격 계수 | `damage.counterCoefficient` | 반격 시 일반 공격 계수 |
| 치명타 배율 | `damage.critMultiplier` | 물리 공격에만 적용 |
| 최대 턴 수 | `maxTurns` | 초과 시 패배 |

### 분노 시스템

| 항목 | 데이터 경로 |
|------|-----------|
| 최대 분노 | `rage.maxRage` |

분노 축적량은 유닛별로 다름:
- 플레이어: `active-skill-tier.data.json`의 `rage_accumulate` 스킬 티어 데이터 `amount`
- 적: `enemy.data.json`의 각 적 `ragePerAttack`

### 적 생성

| 항목 | 데이터 경로 |
|------|-----------|
| 2체 동시 출현 확률 | `enemy.dualSpawnChance` |
| 2체 시 스탯 배율 | `enemy.dualStatMultiplier` |
| 챕터당 스케일링 | `enemy.scalingPerChapter` |
| 탑 층당 스케일링 | `enemy.scalingPerTowerFloor` |

### 플레이어 기본 스탯

| 항목 | 데이터 경로 |
|------|-----------|
| HP / ATK / DEF | `playerBaseStats` |

### 일일 리셋 지급

| 항목 | 데이터 경로 |
|------|-----------|
| 도전권 / 곡괭이 | `dailyReset` |

## 2. 적 스탯 (기본값)

> `enemy.data.json`

5개 테마 x (일반 5종 + 엘리트 2종 + 보스 3종) = 50종 + 던전 전용 3종.

모든 적의 기본 스탯(HP, ATK, DEF), 스킬 구성, 분노 축적량은 `enemy.data.json` 참조.

챕터별 적 스탯은 기본 스탯에 `battle.data.json`의 `enemy.scalingPerChapter` 배율이 적용되어 기하급수적으로 상승한다.

## 3. 인카운터

> `encounter.data.json`

### 인카운터 가중치

| 항목 | 데이터 경로 |
|------|-----------|
| 챕터 유형별 가중치 | `weights` (COMBAT/DEMON/CHANCE) |
| 우연 세부 가중치 | `chance.subWeights` (skillBox/spring/blessing/skillSwap) |

### 강제 전투일

| 항목 | 데이터 경로 |
|------|-----------|
| 엘리트/중간보스 확정일 | `forcedBattleDays` |
| 선택적 엘리트 (확률) | `optionalEliteDays` |

### 카운터 임계값

| 항목 | 데이터 경로 |
|------|-----------|
| 중박/대박 임계값 | `counterThreshold` |

### 인카운터 보상

| 항목 | 데이터 경로 |
|------|-----------|
| 악마 체력 대가 | `demon.hpCostPercent` |
| 치유의 샘 회복 | `chance.springHealPercent` |
| 축복 골드 | `chance.blessingGoldBase`, `chance.blessingGoldPerChapter` |
| 중박 룰렛 회복 | `jungbakRoulette.healPercent` |
| 중박 룰렛 골드 | `jungbakRoulette.goldAmount` |
| 대박 룰렛 확률 | `daebakRoulette.normalRate/angelRate/demonRate` |

### 리롤

| 항목 | 데이터 경로 |
|------|-----------|
| 세션당 리롤 횟수 | `rerollsPerSession` |

### 챕터 클리어 보상

| 항목 | 데이터 경로 |
|------|-----------|
| 골드 / 보석 공식 | `chapterClearReward` |

### 전투 골드 보상 (인카운터당)

| 항목 | 데이터 경로 |
|------|-----------|
| 기본 / 챕터당 / 일차당 | `battle.data.json`의 `combatGoldReward` |

## 4. 장비

### 기본 스탯

> `equipment-base-stats.data.json`

슬롯별/등급별 기본 스탯(ATK, HP) 전체 테이블은 데이터 파일 참조.

### 강화

> `equipment-constants.data.json`

| 항목 | 데이터 경로 |
|------|-----------|
| 레벨당 고정 스탯 증가 | `upgradeFlatPerLevel` |
| 승급 레벨 | `promoteLevels` |
| 기본 강화 비용 | `upgradeCostBase` |
| 구간별 배율 | `upgradeCostTiers` |

### 합성

| 항목 | 데이터 경로 |
|------|-----------|
| 등급별 합성 필요 수량 | `mergeCount` |
| mergeLevel 최대 | `mergeEnhanceMax` |
| 고등급 합성 대상 | `highGradeMergeGrades` |

### 장비 판매가

> `equipment-labels.data.json`의 `sellPrices` 참조.

### 장비 패시브 (삭제됨)

> 장비 패시브 시스템은 K-15 작업에서 삭제되었다 (`EquipmentPassiveTable` + `equipment-passive.data.json` 삭제).
> 현재 장비는 기본 스탯 + 부스탯 + 강화/승급으로만 전투력을 제공한다.
>
> **밸런스 영향 분석:**
> - 삭제 전: 슬롯별 패시브가 빌드 다양성에 기여 (검=ATK, 지팡이=마법계수, 활=분노 등)
> - 삭제 후: 무기 종류 간 차별화 요소가 기본 스탯 차이로만 한정됨
> - 검/지팡이/활의 전략적 선택 의미가 약화될 수 있음
> - 향후 무기별 차별화가 필요하면 패시브 시스템 재도입 또는 대체 메카닉 검토 권장

## 5. 뽑기 (가챠)

> `gacha.data.json`

### 장비 뽑기

| 항목 | 데이터 경로 |
|------|-----------|
| 등급별 가중치 | `equipment.gradeWeights` |
| 1회 비용 | `equipment.costPerPull` |
| 신화 확정 횟수 | `equipment.pityThreshold` |
| S등급 확률 | `equipment.sRate` |
| S등급 대상 등급 | `equipment.sEligibleGrades` |

10연차 비용 = 1회 비용 x 9 (1회 무료).

### 펫 뽑기

| 항목 | 데이터 경로 |
|------|-----------|
| 1회 비용 | `pet.costPerPull` |
| 보상 구성 (알/사료) | `pet.eggAmount`, `pet.foodMin`, `pet.foodMax` |

## 6. 스킬 티어별 수치

### 액티브 스킬

> `active-skill-tier.data.json`

모든 액티브 스킬의 T1~T4 수치(계수, 확률, 소환 횟수, 지속 턴 등)는 데이터 파일 참조.

### 패시브 스킬

> `passive-skill-tier.data.json`

모든 패시브 스킬의 T1~T4 수치(흡혈률, 반격 확률, 마스터리 보너스 등)는 데이터 파일 참조.

---

## 7. 펫

> `pet.data.json`

펫별 기본 스탯(ATK, HP), 특수 능력, 등급 배율, 부화 가중치 등 전체 데이터는 파일 참조.

---

## 8. 재능 & 전승

### 재능

> `talent.data.json`

| 항목 | 데이터 경로 |
|------|-----------|
| 스탯별 레벨당 증가량 | `statPerLevel` |
| 등급 구성 (서브등급 수, 총 레벨) | `gradeConfig` |
| 강화 비용 기본/성장률 | `gradeConfig.baseCost`, `gradeConfig.costGrowth` |
| 마일스톤 설정 | `milestoneConfig` |
| 메인 등급 전환 보너스 | `mainGradeBonuses` |
| 서브 등급 전환 보너스 | `gradeConfig.tierBonusBase/tierBonusGrowth` |

### 전승

> `heritage.data.json`

전승 루트별 레벨당 스탯 증가(ATK/DEF/HP/CRIT), 비용, 시너지 계수 등은 데이터 파일 참조.
