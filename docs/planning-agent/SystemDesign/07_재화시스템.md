# 카피바라고! 역기획서 - 재화 시스템

## 재화 전체 맵

```
[프리미엄 재화]          [기본 재화]           [특수 재화]
    보석 ◆                골드 ●               도전권
                          스태미나 ▲            입장권(미구현)
                                               곡괭이

[성장 재화]              [장비 재화]
    책 (4종)              장비 강화석
                          도면(Power Stone)

[가챠 재화]              [콘텐츠 재화]
    은열쇠                   행운 코인(미구현)
    금열쇠
```

## 재화 상세

### 골드

| 항목 | 내용 |
|------|------|
| 용도 | 특성 강화, 전승 강화 |
| 중요도 | 최상 (모든 영구 성장의 기반) |
| 주요 획득 | 탐방(Travel), 챕터 클리어 |
| 보조 획득 | 던전, 이벤트 |
| 소모 속도 | 매우 빠름 (항상 부족) |

### 보석 (Gems)

| 항목 | 내용 |
|------|------|
| 용도 | 보물상자 소환, 경기장 재도전, 일부 구매 |
| 중요도 | 최상 (프리미엄 재화) |
| 무과금 획득 | 던전(드래곤 둥지), 이벤트, 일일 퀘스트 |
| 과금 획득 | 직접 구매 |
| 관리 | 한정 상자 천장 대비 비축 필요 |

### 스태미나

| 항목 | 내용 |
|------|------|
| 용도 | 메인 챕터 진입, 탐방 |
| 중요도 | 높음 (콘텐츠 진입 게이트) |
| 회복 방식 | 시간 경과 자동 회복 |
| 추가 획득 | 광고 시청, 거대 드래곤 둥지, 과금 |
| 소모 전략 | 메인 챕터 우선 → 잔여분 탐방 |

### 도전권

| 항목 | 내용 |
|------|------|
| 용도 | 도전의 타워 진입 |
| 중요도 | 중상 (도면 획득 경로) |
| 획득 | 일일 지급, 광고 시청 |
| 특징 | 실패 시 미소모 |

### 입장권 **(미구현 -- ARENA_TOKEN 미등록)**

| 항목 | 내용 |
|------|------|
| 용도 | PvP 경기장 진입 |
| 중요도 | 중간 |
| 획득 | 일일 지급 |

### 곡괭이

| 항목 | 내용 |
|------|------|
| 용도 | 고블린 광부 채굴 |
| 중요도 | 낮음 |
| 획득 | 일일 지급 |

### 전용 책 (4종)

| 책 종류 | 용도 | 획득 |
|---------|------|------|
| 뼈 책 | 해골 계승 강화 | 콘텐츠 보상 |
| 기사 책 | 기사 계승 강화 | 콘텐츠 보상 |
| 레인저 책 | 유협 계승 강화 | 콘텐츠 보상 |
| 영혼 책 | 유령 계승 강화 | 콘텐츠 보상 |

### 장비 강화석

| 항목 | 내용 |
|------|------|
| 용도 | 장비 레벨업 |
| 중요도 | 높음 |
| 획득 | 콘텐츠 보상, 보물상자 |

### 도면 (Power Stone)

| 항목 | 내용 |
|------|------|
| 용도 | 장비 승급 (10/20/30단계) |
| 중요도 | 높음 (장비 성장 병목) |
| 획득 | 도전의 타워 5/10단계 보상 |

### 은열쇠 (Silver Key)

| 항목 | 내용 |
|------|------|
| 용도 | 모험가 상자 뽑기 |
| 중요도 | 중간 (저등급 장비 획득 수단) |
| 획득 | 챕터 클리어, 일일 퀘스트, 던전 보상 |
| 특징 | 일반~우수 장비만 출현하는 모험가 상자 개봉에 사용 |

### 금열쇠 (Gold Key)

| 항목 | 내용 |
|------|------|
| 용도 | 영웅상자 뽑기 |
| 중요도 | 중상 (중등급 장비 획득 수단) |
| 획득 | 이벤트 보상, 던전 보상, 도전의 타워 보상 |
| 특징 | 우수~에픽 장비가 출현하는 영웅상자 개봉에 사용 |

### 행운 코인 **(미구현 -- LUCKY_COIN 미등록)**

| 항목 | 내용 |
|------|------|
| 용도 | 30일 챕터 행운 머신 이용 |
| 획득 | 행운 상인 (30일 챕터 내) |
| 특징 | 챕터 내 한정 재화 |

## 재화 흐름도

```
[콘텐츠 진행]
    ├→ 골드 → 특성 강화 / 전승 강화
    ├→ 보석 → 보석 장비뽑기/우수 펫뽑기 → 장비/펫
    ├→ 은열쇠 → 모험가 상자 → 일반~우수 장비
    ├→ 금열쇠 → 영웅상자 → 우수~에픽 장비
    ├→ 장비 강화석 → 장비 레벨업
    ├→ 도면 → 장비 승급
    ├→ 전용 책 → 전승 강화
    └→ 펫 알/사료 → 펫 육성

[시간 경과]
    └→ 스태미나 회복 → 콘텐츠 진행 (루프)

[과금]
    ├→ 보석 직접 구매
    ├→ 월정액 (탐방 최대 배속)
    └→ 보물상자 키 구매 (미구현)
```

## 퀘스트 보상

퀘스트 목록, 목표 횟수, 보상 종류/수량은 `quest.data.json` 참조.

### 일일 퀘스트

일일 리셋되는 반복 퀘스트. 챕터 클리어, 던전 완료, 탑 도전 등. 미구현 항목(아레나, 여행) 포함.

### 주간 퀘스트

주간 리셋되는 장기 목표 퀘스트. 챕터 클리어, 장비 뽑기, 장비 판매, 탑 도전 등.

## 7일 출석 보상

일차별 보상 구성은 `attendance.data.json` 참조.

7일 주기, 점진적으로 좋은 보상 지급. 7일 완료 시 자동 리셋
