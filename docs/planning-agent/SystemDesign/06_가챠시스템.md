# 카피바라고! 역기획서 - 가챠(보물상자) 시스템

## 개요

카피바라고의 가챠는 **장비 뽑기**와 **펫 뽑기** 시스템으로 운영된다.
합성 비율 기반의 정직한 확률 분배를 사용한다.

## 보물상자 종류

### 장비 뽑기 (3종)

| 상자 | 드롭 등급 범위 | 사용 재화 | S등급 출현 | 천장 |
|------|---------------|----------|-----------|------|
| 보석 장비뽑기 | 일반~신화 | 보석 | O | O |
| 모험가 상자 | 일반~우수 | 은열쇠 | X | X |
| 영웅상자 | 우수~에픽 | 금열쇠 | X | X |

### 펫 뽑기 (2종)

| 상자 | 보상 | 사용 재화 | 천장 |
|------|------|----------|------|
| 우수 펫뽑기 | 펫 알 + 펫 사료 (많음) | 보석 | X |
| 일반 펫뽑기 | 펫 알 + 펫 사료 (적음) | 펫 알 | X |

---

## 보석 장비뽑기

프리미엄 장비 뽑기. 일반~신화 전 등급이 출현하며, S등급 장비도 획득 가능하다.

### 확률

합성 비율 기반 — 각 등급은 이전 등급 대비 일정 배율로 희귀하다.
등급별 가중치와 확률은 `gacha.data.json`의 `equipment.gradeWeights` 참조.

| 등급 | 가중치 | 확률 (근사) |
|------|--------|------------|
| 일반 | 260 | 68.2% |
| 우수 | 81 | 21.3% |
| 희귀 | 27 | 7.1% |
| 에픽 | 9 | 2.4% |
| 전설 | 3 | 0.8% |
| 신화 | 1 | 0.3% |

기대값: 뽑기 1회 = 모든 등급이 동일한 기대값을 기여하도록 설계.

### 비용

| 소환 유형 | 비용 |
|-----------|------|
| 단일 소환 | 보석 150개 (`equipment.costPerPull`) |
| 10연차 | 보석 1,350개 (단일 비용 x 9, 1회 무료) |

### 신화 확정 시스템 (천장)

일정 횟수 이상 소환 시 신화 장비가 확정으로 지급된다.
확정 횟수는 `gacha.data.json`의 `equipment.pityThreshold` 참조 (현재 180회).

```
[소환 카운트]
0회 ────────────────────── 확정 횟수(180)
│                           │
│  합성 비율 기반 확률 드롭    │
│                           │
└──────────────────────── 신화 확정
```

### S-등급 시스템

| 항목 | 내용 |
|------|------|
| 대상 등급 | 에픽 이상 (`equipment.sEligibleGrades`) |
| 확률 | 2% (`equipment.sRate`) |
| 특성 | 판매 불가, 합성 재료 불가 |

---

## 모험가 상자 뽑기

저등급 장비를 획득하는 무료/저비용 뽑기. 은열쇠로 뽑기 가능하다.

### 확률

| 등급 | 가중치 | 확률 (근사) |
|------|--------|------------|
| 일반 | 3 | 75.0% |
| 우수 | 1 | 25.0% |

### 비용

| 소환 유형 | 비용 |
|-----------|------|
| 단일 소환 | 은열쇠 1개 (`adventurerChest.costPerPull`) |
| 10연차 | 은열쇠 9개 (1회 무료) |

### 제한

- S등급 출현 없음
- 천장 시스템 없음

---

## 영웅상자 뽑기

중등급 장비를 획득하는 뽑기. 금열쇠로 뽑기 가능하다.

### 확률

| 등급 | 가중치 | 확률 (근사) |
|------|--------|------------|
| 우수 | 9 | 69.2% |
| 희귀 | 3 | 23.1% |
| 에픽 | 1 | 7.7% |

### 비용

| 소환 유형 | 비용 |
|-----------|------|
| 단일 소환 | 금열쇠 1개 (`heroChest.costPerPull`) |
| 10연차 | 금열쇠 9개 (1회 무료) |

### 제한

- S등급 출현 없음
- 천장 시스템 없음

---

## 우수 펫뽑기

프리미엄 펫 뽑기. 보석으로 뽑기 가능하며, 펫 알과 사료를 획득한다.

| 항목 | 내용 |
|------|------|
| 비용 | 보석 298개 (`pet.costPerPull`) |
| 보상 | 펫 알 1개 + 펫 사료 1~5개 (`pet` 설정 참조) |
| 천장 | 없음 |

## 일반 펫뽑기

펫 알을 소모하여 뽑는 저비용 펫 뽑기.

| 항목 | 내용 |
|------|------|
| 비용 | 펫 알 1개 (`basicPet.costPerPull`) |
| 보상 | 펫 알 1개 + 펫 사료 0~2개 (`basicPet` 설정 참조) |
| 천장 | 없음 |

---

## 데이터 파일

| 데이터 | 파일 | 키 |
|--------|------|----|
| 보석 장비뽑기 | `gacha.data.json` | `equipment` |
| 모험가 상자 | `gacha.data.json` | `adventurerChest` |
| 영웅상자 | `gacha.data.json` | `heroChest` |
| 우수 펫뽑기 | `gacha.data.json` | `pet` |
| 일반 펫뽑기 | `gacha.data.json` | `basicPet` |
| 장비 등급 라벨/판매가 | `equipment-labels.data.json` | — |
