# 카피바라고! 역기획서 - 가챠(보물상자) 시스템

## 개요

카피바라고의 가챠는 **장비 뽑기** 시스템으로 운영된다.
합성 비율 기반의 정직한 확률 분배를 사용한다.

## 보물상자 종류

| 상자 | 주요 보상 |
|------|-----------|
| 장비 상자 | 일반~신화 장비 |
| 펫 상자 | 펫 알, 펫 사료 |

## 장비 상자 확률

합성 비율 기반 — 각 등급은 이전 등급 대비 일정 배율로 희귀하다.
등급별 가중치와 확률은 `gacha.data.json` 참조.

기대값: 뽑기 1회 = 모든 등급이 동일한 기대값을 기여하도록 설계.

## 소환 비용

| 소환 유형 | 설명 |
|-----------|------|
| 단일 소환 | 1회 비용은 `gacha.data.json` 참조 |
| 10연차 | 단일 소환 비용 × 9 (1회 무료) |

## 신화 확정 시스템

일정 횟수 이상 소환 시 신화 장비가 확정으로 지급된다.
확정 횟수는 `gacha.data.json`의 `pityThreshold` 참조.

```
[소환 카운트]
0회 ────────────────────── 확정 횟수
│                           │
│  합성 비율 기반 확률 드롭    │
│                           │
└──────────────────────── 신화 확정
```

## S-등급 시스템

| 항목 | 내용 |
|------|------|
| 대상 등급 | 에픽 이상 (`gacha.data.json`의 `sEligibleGrades` 참조) |
| 확률 | `gacha.data.json`의 `sRate` 참조 |
| 특성 | 판매 불가, 합성 재료 불가 |

## 펫 상자

| 항목 | 내용 |
|------|------|
| 비용 | `gacha.data.json`의 `pet.costPerPull` 참조 |
| 보상 | 펫 알 + 펫 사료 (수량은 `gacha.data.json`의 `pet` 설정 참조) |
| 천장 | 없음 |

## 데이터 파일

| 데이터 | 파일 |
|--------|------|
| 뽑기 설정/확률 | `gacha.data.json` |
| 장비 등급 라벨/판매가 | `equipment-labels.data.json` |
