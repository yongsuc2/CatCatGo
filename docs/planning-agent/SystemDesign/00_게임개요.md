# 카피바라고! 역기획서 - 게임 개요

## 기본 정보

| 항목 | 내용 |
|------|------|
| 게임명 | 카피바라 고! (Capybara Go!) |
| 개발사 | Habby |
| 장르 | 로그라이크 + 턴제 RPG |
| 플랫폼 | Android, iOS |
| 출시일 | 2024년 10월 20일 |
| 시점 | 2D 종스크롤 |
| 조작 | 터치 기반 선택형 (자동 전투) |

## 핵심 게임플레이 루프

```
[스태미나 소모] → [챕터 진입] → [인카운터 선택] → [턴제 전투] → [스킬 획득]
     ↑                                                              ↓
 [재화 획득] ← [챕터 클리어] ← [보스 전투] ← [능력 강화(스킬 조합)]
     ↓
[성장 시스템 투자] → [특성 강화 / 장비 강화 / 전승 강화 / 펫 육성]
     ↓
[더 높은 챕터 도전] → [반복]
```

## 핵심 루프 상세

### 1차 루프: 챕터 진행 (세션 단위)
1. 스태미나를 소모하여 챕터 진입 (소모량은 게임 설정 참조)
2. 챕터 내에서 다양한 인카운터(이벤트 노드) 발생
3. 인카운터 선택에 따라 스킬 획득, 전투, 회복 등 진행
4. 턴제 자동 전투로 적과 교전
5. 챕터 보스 격파 시 클리어, 보상 획득

### 2차 루프: 자원 축적 (일일 단위)
1. 메인 챕터 진행으로 골드, 보석, 장비 획득
2. 탐방(Travel) 모드로 클리어 챕터 자동 반복 파밍
3. 일일 던전(드래곤 둥지, 아스트랄 나무, 환검의 하늘섬) 소화
4. 도전 콘텐츠(타워, 지하감옥) 진행
5. PvP 경기장 도전

### 3차 루프: 영구 성장 (장기 단위)
1. 골드로 특성(Talent) 강화 → 기본 스탯 상승
2. 장비 강화/합성으로 전투력 상승
3. 전승(Heritage) 시스템으로 빌드 특화
4. 펫 육성으로 패시브 효과 확보

## 챕터 유형

현재 구현된 챕터는 **60일 챕터** 1종이다.

| 유형 | 기간 | 특징 | 상태 |
|------|------|------|------|
| 60일 챕터 | 60턴 | 중박/대박 룰렛 시스템 포함 | 구현 완료 |
| 30일 챕터 | 30턴 | 행운 상인 + 행운 코인 머신 (100%/70%/40% 성공률 선택) | 미구현 |
| 5일 챕터 | 5턴 | 웨이브 형식 전투, 누적 턴 시스템 | 미구현 |

## 인카운터 유형

| 인카운터 | 설명 |
|----------|------|
| 전투 (COMBAT) | 적과 자동 전투, 승리 시 골드 획득 |
| 악마 (DEMON) | 최대 HP 대가로 랜덤 스킬 획득 |
| 우연 (CHANCE) | 스킬 상자 / 치유의 샘 / 축복(골드) / 스킬 교환 중 가중치 기반 랜덤 |
| 중박 룰렛 | 중박 카운터 임계값 도달 시 발동 (체력 회복 / 스킬 / 골드 선택) |
| 대박 룰렛 | 대박 카운터 임계값 도달 시 발동 (신화 스킬 / 천사의 힘 / 악마의 힘) |

## 탐방(Travel) 시스템

| 항목 | 내용 |
|------|------|
| 진입 조건 | 스태미나 소모 |
| 대상 | 최고 클리어 챕터 자동 반복 |
| 배율 | `battle.data.json` 또는 탐방 설정 데이터 참조 (최대 배율은 월정액 필요) |
| 주요 보상 | 골드 |
| 위치 | 메인 재화 파밍 수단 |
